# Daily KPI Tracker Guide (Traffic Growth Focus)

Update this once per day (ideally 08:45-09:00 local):

- `organic_sessions`: from GA4 (organic only)
- `gsc_clicks`, `gsc_impressions`, `avg_position`: from Google Search Console (last complete day)
- `indexed_pages`: site: query + GSC index report snapshot
- `new_pages_published`: count of newly deployed indexable pages
- `top_landing_page`: highest organic entry route
- `landing_page_clicks`: clicks for top landing page
- `waitlist_signups`: number of new email/interest submissions
- `founding_partner_leads`: qualified B2B monetization interest
- `notes`: major changes, distribution pushes, algorithm/weather notes

Do not fabricate numbers; leave blank if unavailable.
